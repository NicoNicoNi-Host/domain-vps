# domain-vps
二级域名-vps用

基于重定向,仅为了方便记忆网站;若想实际为vps添加域名,可以使用宝塔面板



## CentOS

| 网站     | URL                      | 重定向                                  |
| -------- | ------------------------ | --------------------------------------- |
| 宝塔面板 | https://vps.niconicoli.com/bt    | http://150.230.108.224:8888/36b59849    |
| zfile    | https://vps.niconicoli.com/zfile | http://150.230.108.224:8080/2/main/root |
|          |                          |                                         |
